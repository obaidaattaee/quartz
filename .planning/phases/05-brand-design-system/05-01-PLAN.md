**Step 1: Download Space Grotesk WOFF2 to resources/fonts/ (per D-24, D-27, BRAND-07)**

D-24 locks the decision: "Both committed to resources/fonts/ and loaded via @font-face in app.css." D-27: "Font files... all local [committed to repo]." Fontsource puts files in node_modules/ which is gitignored -- that violates D-24/D-27.

```bash
mkdir -p resources/fonts
```

Download the Space Grotesk variable font WOFF2 from Google Fonts API. The variable font file covers all weights (300-700) in a single WOFF2:

```bash
curl -L "https://fonts.google.com/download?family=Space+Grotesk" -o /tmp/space-grotesk.zip
unzip -o /tmp/space-grotesk.zip -d /tmp/space-grotesk/
```

Then find the variable WOFF2 file. Google Fonts packages include TTF files. To get WOFF2, use the Google Fonts CSS API to extract the WOFF2 URL:

```bash
curl -s -H "User-Agent: Mozilla/5.0 (Macintosh)" \
  "https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@300..700&display=swap" \
  | grep -oP 'url\(\K[^)]+\.woff2' | head -1
```

Download that URL and save to `resources/fonts/SpaceGrotesk-Variable.woff2`. If the grep approach fails, an alternative is to use `fontsource` temporarily to get the file, then copy it:

```bash
# Fallback approach if direct download is tricky:
npx -y @fontsource-variable/space-grotesk 2>/dev/null || true
find node_modules/@fontsource-variable/space-grotesk -name "*.woff2" -path "*latin*" | head -1 | xargs -I{} cp {} resources/fonts/SpaceGrotesk-Variable.woff2
```

Verify the WOFF2 file exists and is >10KB (variable fonts are typically 20-50KB):
```bash
ls -la resources/fonts/SpaceGrotesk-Variable.woff2
```

If neither approach yields a WOFF2 file, download the TTF from Google Fonts and convert:
```bash
# Last resort: download TTF and convert with woff2 tool if available
# pip install fonttools brotli && python -c "from fontTools.ttLib import TTFont; f=TTFont('/tmp/space-grotesk/SpaceGrotesk-VariableFont_wght.ttf'); f.flavor='woff2'; f.save('resources/fonts/SpaceGrotesk-Variable.woff2')"
```

The key requirement: `resources/fonts/SpaceGrotesk-Variable.woff2` must exist and be a valid WOFF2 variable font file. This file will be git-committed.
