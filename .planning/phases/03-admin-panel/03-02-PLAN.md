    File: `resources/js/components/admin/tiptap-editor.tsx`
    - Props: `{ content: string; onChange: (html: string) => void; direction?: 'ltr' | 'rtl'; placeholder?: string; className?: string }`
    - Uses `useEditor` from `@tiptap/react`:
      ```
      const editor = useEditor({
          extensions: [
              StarterKit.configure({
                  heading: { levels: [2, 3, 4] },
              }),
              Image.configure({
                  HTMLAttributes: { class: 'max-w-full h-auto rounded-lg' },

          immediatelyRender: false,
          onUpdate: ({ editor }) => {

          },
      });
      ```
    - NOTE on textDirection: Tiptap v3 `textDirection` option is set at the editor level. Pass `textDirection: direction` in the useEditor config if the prop is `'rtl'` or `'auto'`. If Tiptap types don't accept it, apply via CSS `dir` attribute on the editor container instead.

              dir={direction}

      - `const hasArErrors = errors && Object.keys(errors).some(k => k.endsWith('_ar'));`
