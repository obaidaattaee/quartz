    **Step 3: Create BlogPostController (per D-03, D-04)**

    File: `app/Http/Controllers/Admin/BlogPostController.php`
    - Namespace: `App\Http\Controllers\Admin`
    - `index(Request $request)`:
      - Query: `BlogPost::with('featuredImage')->when($request->search, fn ($q, $s) => $q->where('title_en', 'like', "%{$s}%")->orWhere('title_ar', 'like', "%{$s}%"))->when($request->status, fn ($q, $s) => $q->where('status', $s))->latest()->paginate(15)->withQueryString()`
      - Return: `Inertia::render('admin/blog/index', ['posts' => $posts, 'filters' => $request->only('search', 'status')])`
    - `create()`:
      - Return: `Inertia::render('admin/blog/create')`
    - `store(StoreBlogPostRequest $request)`:
      - Data: `$request->validated()` + `['author_id' => $request->user()->id, 'published_at' => $request->status === 'published' ? now() : null]`
      - `BlogPost::create($data)`
      - Return: `redirect()->route('admin.blog.index')->with('success', 'Blog post created.')`
    - `edit(BlogPost $blog)`:
      - Load: `$blog->load('featuredImage')`
      - Return: `Inertia::render('admin/blog/edit', ['post' => $blog])`
    - `update(UpdateBlogPostRequest $request, BlogPost $blog)`:
      - Data: `$request->validated()` + published_at logic (set to now() on first publish, keep existing if already set)
      - `$blog->update($data)`
      - Return: `redirect()->route('admin.blog.index')->with('success', 'Blog post updated.')`
    - `destroy(BlogPost $blog)`:
      - `$blog->delete()`
      - Return: `redirect()->route('admin.blog.index')->with('success', 'Blog post deleted.')`
