    **Create app/Http/Controllers/BlogController.php:**

    A public controller with 5 methods. All methods follow the established pattern: locale prefix, eager loading, SeoService for meta tags, Inertia::render with withViewData.

    **Method 1: index(string $locale)**
    - Query: `BlogPost::published()->with(['author', 'author.avatar', 'featuredImage', 'categories'])->latest('published_at')->paginate(6)->withQueryString()`
    - Transform each post to include `reading_time_en` (calculated via `strip_tags($post->content_en)` word count / 200, ceil, min 1) and `reading_time_ar` (same logic with content_ar, / 180)
    - Load all categories for the filter tabs: `Category::orderBy('sort_order')->get()`
    - SEO: `SeoService::forStaticPage('blog', $locale, 'Blog', "/{$locale}/blog")`
    - Render: `Inertia::render('public/blog/index', ['posts' => $posts, 'categories' => $categories, 'seo' => $seo])->withViewData(['seo' => $seo])`

    **Method 2: show(string $locale, string $slug)**
    - Query: `BlogPost::published()->with(['author', 'author.avatar', 'featuredImage', 'ogImage', 'categories', 'tags'])->where('slug', $slug)->firstOrFail()`
    - Add reading time fields to post (same calculation as index)
    - Related posts logic (per D-09): Get post's category IDs, find up to 3 published posts in same categories (excluding current), ordered by published_at desc. If fewer than 3, fill remaining with posts sharing tags. If still fewer than 3, fill with most recent posts. Each related post loaded with ['author', 'author.avatar', 'featuredImage', 'categories'] and reading time added.
    - SEO: `SeoService::forBlogPost($post, $locale)`
    - Render: `Inertia::render('public/blog/show', ['post' => $post, 'relatedPosts' => $relatedPosts, 'seo' => $seo])->withViewData(['seo' => $seo])`

    **Method 3: category(string $locale, string $slug)**
    - Find category: `Category::where('slug', $slug)->firstOrFail()`
    - Query: Same as index but filtered by category via `whereHas('categories', fn($q) => $q->where('categories.id', $category->id))`
    - Load all categories for filter tabs
    - SEO: `SeoService::forStaticPage('blog', $locale, $locale === 'ar' ? $category->name_ar : $category->name_en, "/{$locale}/blog/category/{$slug}")`
    - Render: `Inertia::render('public/blog/category', ['posts' => $posts, 'categories' => $categories, 'category' => $category, 'seo' => $seo])->withViewData(['seo' => $seo])`

    **Method 4: tag(string $locale, string $slug)**
    - Find tag: `Tag::where('slug', $slug)->firstOrFail()`
    - Query: Same as index but filtered by tag via `whereHas('tags', fn($q) => $q->where('tags.id', $tag->id))`
    - SEO with tag name as title
    - Render: `Inertia::render('public/blog/tag', ['posts' => $posts, 'tag' => $tag, 'seo' => $seo])->withViewData(['seo' => $seo])`

    **Method 5: author(string $locale, int $id)**
    - Find author: `User::with('avatar')->findOrFail($id)`
    - Query: Same as index but filtered by `where('author_id', $id)`
    - SEO with author name as title
    - Render: `Inertia::render('public/blog/author', ['posts' => $posts, 'author' => $author, 'seo' => $seo])->withViewData(['seo' => $seo])`

    **Helper method: addReadingTime(BlogPost $post): BlogPost**
    Private method that calculates reading time for both locales and adds as attributes:
    ```php
    private function addReadingTime(BlogPost $post): BlogPost
    {
        $post->reading_time_en = max(1, (int) ceil(str_word_count(strip_tags($post->content_en)) / 200));
        $post->reading_time_ar = max(1, (int) ceil(str_word_count(strip_tags($post->content_ar)) / 180));
        return $post;
    }
    ```

    Route::get('/blog/{slug}', [BlogController::class, 'show'])->name('blog.show');

    Note: The `{slug}` route must come AFTER the named routes (category, tag, author) to avoid conflicts.
