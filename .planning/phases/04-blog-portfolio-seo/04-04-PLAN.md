    **Create app/Http/Controllers/Admin/CategoryController.php:**
    Standard admin CRUD for categories (per D-05). Categories are admin-managed, not user-created.

    - `index()`: Load `Category::withCount('posts')->orderBy('sort_order')->get()`, render `admin/categories/index`
    - `store(Request $request)`: Validate name_en (required|string|max:100), name_ar (required|string|max:100), slug (required|string|unique:categories), sort_order (integer|min:0). Create category. Redirect to index with success.
    - `update(Request $request, Category $category)`: Same validation (slug unique except current). Update. Redirect.
    - `destroy(Category $category)`: Delete. Redirect. Posts are not deleted (pivot entries cascade).

    Manages SEO metadata for static pages.

    - `update(Request $request)`: Receives an array of SEO data keyed by page_key. For each page_key, `SeoMetadata::updateOrCreate(['page_key' => $key], $validated)`. Redirect with success.

    Note: This is a NEW SeoSettingController specifically for static page SEO. The existing SiteSettingController handles branding/contact info. Add this route under admin-only middleware.
